#  Flood Warning System

A simple web demo for a real-time flood warning system that simulates rainfall and river level monitoring, predicts overflow risk, and shows alerts based on sensor data.

 Live Demo: https://24maindes.github.io/flood_warning_system/

## Features
- Simulate rainfall and river level input  
- Prediction status based on thresholds  
- Alert log for potential flood risk  
- Simple UI to demonstrate flow from sensors → prediction → alerts

## Tech Stack
- HTML  
- CSS  
- JavaScript

## Usage
Open the live demo, adjust rainfall/river level, and view prediction and alerts.
